# 内容规划.md

## 📖 10卷大纲

### 第一卷：觉醒的囚徒
- **核心主题**：规则认知 + 团队建立
- **副本数**：10个
- **章节**：1-100章
- **目标**：主角团首次相遇，初步了解规则系统，建立信任关系
- **情感爆点**：首次牺牲，团队信任建立
- **关键线索**：发现副本背后的系统存在

### 第二卷：禁忌的迷宫
- **核心主题**：信息收集 + 身份揭露
- **副本数**：10个
- **章节**：101-200章
- **目标**：收集更多副本信息，发现团队成员的秘密
- **情感爆点**：成员身份揭露，冲突爆发
- **关键线索**：发现前通关者的存在

### 第三卷：污染的认知
- **核心主题**：心智侵蚀 + 信任危机
- **副本数**：10个
- **章节**：201-300章
- **目标**：面对规则的认知污染，处理团队信任危机
- **情感爆点**：心智被侵蚀，信任崩塌
- **关键线索**：发现规则可被污染

### 第四卷：规则的猎手
- **核心主题**：主动破局 + 团队分化
- **副本数**：10个
- **章节**：301-400章
- **目标**：主动寻找规则漏洞，团队内部出现分歧
- **情感爆点**：团队分化，面临选择
- **关键线索**：发现规则之间的关联

### 第五卷：元规则的阴影
- **核心主题**：系统奥秘 + 幕后线索

- **章节**：401-500章

- **情感爆点**：世界观颠覆，信念崩塌
- **关键线索**：接触到系统的核心

### 第六卷：死亡的盛宴

## 📝 第一卷详细规划

### 副本1：午夜电梯（1-10章）
- **类型**：生存类
- **场景**：13层电梯，每层一条规则
- **核心规则**：
  1. 电梯只能在整点运行
  2. 每层停留时间为60秒
  3. 每层必须完成指定任务才能进入下一层
  4. 禁止在电梯内交谈
  5. 禁止查看电梯外的景象
- **恐怖元素**：电梯血字、镜中鬼影、坠落感
- **角色出场**：林知远、秦弈、苏晚晴、陈烈、零
- **情节大纲**：
  1. **入局**：五人被强制拉入电梯
  2. **试探**：首次违规，一名龙套角色死亡
  3. **分化**：团队出现信任危机
  4. **突破**：林知远发现规则漏洞
  5. **反转**：发现电梯是垂直的时间线
  6. **破局**：利用时间差破解规则
  7. **余波**：零的身份首次引起怀疑

### 副本2：谎言医院（11-20章）
- **类型**：对抗类
- **场景**：废弃医院，身份隐藏+狼人杀模式
- **核心规则**：
  1. 每个人都有一个隐藏身份
  2. 说谎者死
  3. 必须说谎
  4. 每天必须进行一次身份投票
  5. 被投票出局者死亡
- **恐怖元素**：解剖室阴魂、输液管血线、手术台鬼影
- **角色出场**：核心五人+其他6名玩家
- **情节大纲**：
  1. **入局**：进入医院，获得身份
  2. **试探**：首次投票，一名玩家死亡
  3. **分化**：团队内部出现怀疑
  4. **突破**：苏晚晴通过微表情发现说谎者
  5. **反转**：发现身份是可以交换的
  6. **破局**：利用规则矛盾破解困局
  7. **余波**：发现医院是系统的测试场

### 副本3：时间学校（21-30章）
- **类型**：解谜类
- **场景**：循环时间的学校
- **核心规则**：

- **情节大纲**：

  6. **破局**：利用记忆残留破解循环
  7. **余波**：发现时间循环与系统有关
