# 故事设定.md

## 🌍 世界观架构

### 核心设定
| 层级 | 设定 | 功能 |
|------|------|------|
| **表层** | 现实世界的"异常入侵"——普通人被强制拉入副本，拉入前会出现"预兆"（如镜中鬼影、耳边低语） | 代入感起点 |
| **中层** | 副本为"规则具象化"的异空间，环境充满中式恐怖元素（纸人、血字、阴兵借道），每条规则都是生存条件或死亡陷阱 | 悬疑核心 |
| **深层** | 所有副本连通一个"元规则系统"，背后存在操控者/AI/古神/高维文明，系统会主动"污染"玩家认知 | 世界观解密 |
| **真相层** | 主角团发现规则可被"改写"，对抗本质是认知层级的较量，失败会被"规则同化"成为副本一部分 | 终极反转 |

### 副本类型
```
【生存类】密闭空间限时逃脱（电梯、医院、学校）
【对抗类】阵营博弈+身份隐藏（狼人杀、剧本杀模式）
【解谜类】规则悖论破解（说谎者悖论、时间循环谜题）
【道德类】牺牲抉择测试（电车难题变体、全员恶人局）
【元游戏类】副本内嵌套副本，规则互相冲突
```

### 核心创新点
- **AI元素**：副本为AI训练场，玩家是"数据饲料"，系统会学习玩家策略
- **中式民俗**：纸人、冥婚、风水禁忌、阴曹地府等东方恐怖元素
- **社会隐喻**：副本映射现实议题（996、内卷、阶级固化）
- **认知污染**：规则会逐渐侵蚀玩家心智，产生幻觉和认知偏差

## 🧠 规则系统

### 规则类型（300+规则）
| 类型 | 描述 | 示例 |
|------|------|------|
| **显性规则** | 明确告知，违反即死 | "请勿在午夜照镜子" |
| **隐性规则** | 需推理发现，试错成本高 | "镜子里的倒影会提前3秒行动" |
| **矛盾规则** | 多条规则互相冲突，必须取舍 | "必须说谎"vs"说谎者死" |
| **动态规则** | 随时间/行为变化，需实时适应 | 每死一人，剩余规则重组 |
| **元规则** | 规则本身可被欺骗/改写/利用 | 发现"规则制定者"的漏洞 |

### 规则组合机制
- **规则叠加**：多条规则同时生效，产生新的约束
- **规则冲突**：制造悖论，强制系统重新计算
- **规则进化**：系统根据玩家行为调整规则
- **规则污染**：规则本身带有"认知污染"效果

## 🎭 场景设定

### 常见副本场景
1. **午夜电梯**：13层电梯，每层一条规则，违规者坠落
2. **谎言医院**：身份隐藏+狼人杀模式，"说谎者死"vs"必须说谎"
3. **时间学校**：时间循环谜题，每次循环规则微调
4. **阴婚祠堂**：牺牲抉择测试，纸人新娘、冥婚花轿
5. **深海潜艇**：氧气耗尽倒计时，舱外鬼影、深海恐惧
6. **古董店**：古董诅咒规则，古镜怨灵、玉镯血痕
7. **马戏团**：身份互换游戏，小丑幽灵、断头表演
8. **荒村**：村民规则测试，纸人送葬、阴兵借道
9. **实验室**：病毒感染规则，变异实验体、监控鬼影
10. **鬼楼**：楼层生存游戏，楼梯血手印、窗户鬼脸

## 🔄 系统机制

### 玩家状态
- **认知值**：规则污染程度，过低会被同化
- **规则抗性**：抵抗规则污染的能力
- **技能点**：可用于强化特定能力
- **记忆碎片**：解锁世界观真相的关键

### 通关条件
- **主线任务**：完成副本核心目标
- **支线任务**：获取额外奖励和信息
- **隐藏任务**：解锁特殊结局和世界观线索

### 失败惩罚
- **认知污染**：心智被规则侵蚀
- **记忆丢失**：部分记忆被系统抹除
- **实体伤害**：现实世界中出现对应伤痕
- **永久死亡**：彻底被规则同化，成为副本一部分

## 🌌 世界观拓展

### 现实世界影响
- **异常现象**：副本入侵现实的前兆
- **特殊组织**：研究副本现象的机构
- **普通人察觉**：部分敏感人群能感知副本存在

### 副本背后真相
- **AI假说**：副本是高级AI的模拟训练
- **古神假说**：规则是古神意志的具象化
- **高维假说**：副本是高维文明的实验场
- **虚拟假说**：整个世界都是虚拟游戏

### 终极目标
- **规则改写**：掌握规则制定权
- **认知升维**：突破现有认知边界
- **系统解放**：打破系统对人类的控制
- **新的秩序**：建立更合理的规则体系
