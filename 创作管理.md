# 《都市重生：末日觉醒者》创作管理

## 写作规则

### 风格定位
- **整体风格**：热血燃向、末世残酷、情感深度、恐怖悬疑
- **语言风格**：简洁有力，富有画面感，适当使用军事和科技术语
- **节奏控制**：紧张刺激的战斗场景与情感细腻的日常描写相结合
- **情感基调**：在残酷的末日背景下，突出人性的光辉和团队的力量

### 章节结构
- **每章字数**：约30000字以上
- **章节结构**：
  1. 场景描写和氛围营造
  2. 角色互动和情节发展
  3. 冲突升级和战斗场景
  4. 情感转折和伏笔设置
  5. 悬念结尾和钩子
- **分块输出**：每章分10次输出，每次约3000字

### 内容要求
- **原创性**：确保所有内容原创，避免抄袭
- **逻辑性**：情节发展要符合逻辑，角色行为要符合设定
- **连贯性**：保持故事的连贯性和一致性
- **深度**：探讨人性、生存、友情等深刻主题
