# 创作管理.md

## 📝 写作规则

### 基本要求
- **章节长度**：每章至少3万字
- **输出模式**：每章分15次输出
- **副本密度**：每10章 = 1个副本
- **更新频率**：保持稳定的创作节奏
- **风格统一**：保持悬疑智斗、暗黑心理、认知博弈的风格

### 内容要求
- **规则体系**：每章至少3条新规则，确保规则多样性
- **角色互动**：每章至少5次核心角色互动
- **恐怖元素**：每章至少2次恐怖场景描写
- **智斗情节**：每章至少1次智力高光时刻
- **伏笔设置**：每章至少2个伏笔或悬念

### 质量标准
- **逻辑一致性**：确保情节发展和规则逻辑自洽
- **语言质量**：保持流畅的叙述，避免重复和冗余
- **节奏控制**：合理安排紧张和舒缓的节奏
- **角色塑造**：确保角色行为符合其性格和背景
- **世界观融合**：自然融入世界观设定，避免生硬

## 🔄 创作流程

### 章节创作流程
1. **设定分析**：分析当前副本的规则和设定
2. **情节规划**：规划章节的情节发展
3. **角色安排**：安排角色的出场和互动
4. **规则设计**：设计章节中的规则和挑战
5. **分块创作**：将章节内容分为15次输出
6. **质量检查**：检查章节的逻辑和质量
7. **状态更新**：更新创作状态和进度

### 副本创作流程
1. **副本设定**：确定副本的类型、规则和场景
2. **角色配置**：确定参与副本的角色
3. **情节大纲**：规划副本的整体情节
4. **章节分配**：将副本内容分配到10个章节
5. **规则设计**：设计副本的核心规则和挑战
6. **伏笔设置**：在副本中设置伏笔和悬念
7. **质量检查**：检查副本的逻辑和一致性
8. **状态更新**：更新副本的完成状态

### 卷创作流程
1. **卷主题**：确定卷的核心主题和目标
2. **副本规划**：规划卷内的10个副本
3. **角色发展**：规划角色在卷内的成长
4. **情节串联**：确保副本之间的情节连贯性
5. **伏笔回收**：回收之前设置的伏笔
6. **新伏笔设置**：设置新的伏笔和悬念
7. **质量检查**：检查卷的整体质量
8. **状态更新**：更新卷的完成状态
